Caliban-Partition Model: Interactive Simulation
A browser-based Monte Carlo simulation of the Caliban-Partition Model (CPM), a mathematical framework modelling social exclusion in group conversations using partition theory.
The model predicts that prime-sized groups (5, 7, 11, 13) produce significantly higher rates of member isolation than factorisable groups (4, 6, 8, 10, 12) because prime numbers cannot be split into equal subgroups without leaving someone out. The isolated member is termed the "Caliban," after the sole native of an island in Shakespeare's The Tempest.
Live site: calibanpartitionmodel.com
Simulation modes

Mode I: Animated partition visualisation for a single group size
Mode II: Head-to-head comparison of prime vs factorisable groups
Mode III: Caliban rotation tracker with fairness metrics
Mode IV: Longitudinal study across all group sizes n=3 to 15

Associated paper: The Caliban-Partition Model: A Mathematical Framework for Social Exclusion within Small Groups (working paper, 2025). Contact for reuse or citation.
